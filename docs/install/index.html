Cách 1 — Đưa cho chính AI cài

Dành cho: tất cả mọi người · thời gian: ~5 phút · không cần biết dòng lệnh

Mở AI của bạn, dán nguyên câu này. AI sẽ tải mã nguồn, đọc trình cài trước khi chạy, dựng môi trường, đăng ký vào cả 3 nền tảng và tự học 4 skill + 6 lệnh:

dán vào AI 
Clone https://github.com/ducnguyen221/powerbi-agent
vào ~/.mcp/powerbi-mcp rồi chạy install.ps1 trong đó
(đọc script trước khi chạy), sau đó restart MCP host.

Xong thì khởi động lại AI và gõ /pbi-setup — xem tiếp Hướng dẫn.

Cách 2 — Tự chạy lệnh (git + installer)

Dành cho: người quen dòng lệnh, muốn kiểm soát từng bước

PowerShell 
# 1. Lấy mã nguồn về đúng chỗ khuyến nghị
git clone https://github.com/ducnguyen221/powerbi-agent "$env:USERPROFILE\.mcp\powerbi-mcp"
cd "$env:USERPROFILE\.mcp\powerbi-mcp"
# 2. Cài in-place: venv + dò ADOMD/TOM + đăng ký 3 host + skills + lệnh
powershell -ExecutionPolicy Bypass -File .\install.ps1
# 3. (khuyến nghị) cài kèm modeling chính chủ Microsoft
claude mcp add powerbi-modeling -s user -- npx -y "@microsoft/powerbi-modeling-mcp@latest" --start

Trình cài idempotent (chạy lại an toàn), backup config trước khi sửa, không đụng file bí mật. Tham số: -Hosts claude,codex,antigravity chọn nền tảng · -SkipVenv chỉ cập nhật đăng ký. Bản hướng dẫn chi tiết từng bước + ảnh: INSTALL.html.

Cách 3 — Chỉ cài skills (dạng plugin, không cần Python)

Dành cho: người chỉ cần quy trình & kiến thức (chưa cần AI thao tác trực tiếp lên Power BI)

Claude Code / Codex 
claude plugin marketplace add ducnguyen221/powerbi-agent
claude plugin install powerbi-agent@powerbi-agent

Muốn đủ 16 công cụ (truy vấn, dựng báo cáo…) → quay lại Cách 1/2.

Sau khi cài

Kiểm tra 30 giây

1

Khởi động lại AI

Đóng mở lại Claude Code / Codex / Antigravity để nhận công cụ mới.

2

Kiểm tra kết nối

Claude Code: chạy claude mcp list — phải thấy powerbi-mcp-bridge ✔ Connected.

3

Thử lệnh đầu tiên

Mở 1 file .pbix bất kỳ trong Power BI Desktop rồi hỏi AI: "Liệt kê các báo cáo Power BI đang mở và cho xem danh sách bảng của model." AI trả về danh sách bảng = cài thành công 🎉

4

Thiết lập bộ nhớ tri thức

Gõ /pbi-setup — chỉ định thư mục lưu tri thức dự án (1 lần). Chi tiết: Hướng dẫn.
